# Image Notes

These PNGs are locally generated lifestyle/product placeholders for demos. They intentionally contain no readable text, no real-world logos, no people, and no medical imagery.

Use them as background, atmosphere, or routine context in prototypes. Keep all message copy editable in HTML, slides, docs, or app UI.

## Intended Prompts

- Morning routine setup: mug, journal, phone card, and simple routine objects on a warm surface, no text.
- Mindful content moment: abstract app/content cards, headphones, and soft natural shapes, no text.
- Community prompt moment: friendly non-human visual composition suggesting shared routines, no people or text.
- Commerce routine kit: simple wellness-adjacent products such as journal, bottle, and stretch band, no claims or logos.
